# Problem A12

A **duopoly model** taken from Krawczyk and Uryasev (2000). This is a standard NEP (no coupling constraints). The Jaco-Hessian is constant and symmetric positive definite.

**Players:** $N = 2$, each controlling a single variable $x^\nu \in \mathbb{R}$.

## Objective Functions

$$\theta_\nu(\mathbf{x}) := x^\nu\bigl(\rho(x^1 + x^2) + \lambda - d\bigr), \quad \nu = 1, 2$$

with parameters $d = 20$, $\lambda = 4$, $\rho = 1$.

## Constraints

$$x^\nu \in [-10,\, +10], \quad \nu = 1, 2$$

## Known Solution

$$\bar{x} = \left(\frac{16}{3},\; \frac{16}{3}\right)^T \approx (5.3333,\; 5.3333)^T$$
