# Problem A17

A jointly convex GNEP taken from Nabetani (2008) with **infinitely many solutions**. The Jaco-Hessian is constant, symmetric, and positive definite.

**Players:** $N = 2$. Player 1 controls $x^1 = (x_1, x_2)^T \in \mathbb{R}^2$; player 2 controls $x^2 = x_3 \in \mathbb{R}$.

## Optimization Problems

$$\min_{x_1, x_2}\; x_1^2 + x_1 x_2 + x_2^2 + (x_1+x_2)x_3 - 25x_1 - 38x_2$$

$$\text{s.t.}\quad x_1, x_2 \geq 0, \quad x_1 + 2x_2 - x_3 \leq 14, \quad 3x_1 + 2x_2 + x_3 \leq 30$$

$$\min_{x_3}\; x_3^2 + (x_1+x_2)x_3 - 25x_3$$

$$\text{s.t.}\quad x_3 \geq 0, \quad x_1 + 2x_2 - x_3 \leq 14, \quad 3x_1 + 2x_2 + x_3 \leq 30$$

## Known Solutions

The problem has infinitely many solutions:

$$\left\{\,(\alpha,\; 11-\alpha,\; 8-\alpha)^T \;\middle|\; \alpha \in [0,\, 2]\right\}$$

The normalized equilibrium corresponds to $\alpha = 0$: $\bar{x} = (0,\; 11,\; 8)^T$.
