# Problem A8

A three-player GNEP taken from Facchinei, Fischer, and Piccialli (2009) with **infinitely many solutions**. The Jaco-Hessian is constant and negative semi-definite.

**Players:** $N = 3$, each controlling a single variable $x^\nu \in \mathbb{R}$.

## Optimization Problems

$$\min_{x^1}\; -x^1 \qquad \text{s.t. } x^3 \leq x^1 + x^2 \leq 1,\; x^1 \geq 0$$

$$\min_{x^2}\; (x^2 - 0.5)^2 \qquad \text{s.t. } x^3 \leq x^1 + x^2 \leq 1,\; x^2 \geq 0$$

$$\min_{x^3}\; (x^3 - 1.5\,x^1)^2 \qquad \text{s.t. } 0 \leq x^3 \leq 2$$

## Known Solutions

The problem has infinitely many solutions:

$$\left\{\,(\alpha,\; 1-\alpha,\; 1.5\alpha)^T \;\middle|\; \alpha \in \left[\tfrac{1}{2},\, \tfrac{2}{3}\right]\right\}$$
