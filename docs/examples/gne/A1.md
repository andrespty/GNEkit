# Problem A1

A variant of the internet switching model introduced by Kesselman et al. and further analyzed by Facchinei et al. This version gives the first player a "privileged status" through distinct bound constraints.

**Players:** $N = 10$, each controlling a single variable $x^\nu \in \mathbb{R}$

## Objective Functions

$$\theta_\nu(\mathbf{x}) := \frac{-x^\nu}{x^1 + \cdots + x^N}\left(1 - \frac{x^1 + \cdots + x^N}{B}\right), \quad \forall\, \nu = 1, \ldots, N$$

with $B = 1$.

## Constraints

**Player 1:**

$$0.3 \leq x^1 \leq 0.5$$

**Players $2, \ldots, 10$:**

$$x^1 + \cdots + x^N \leq B, \quad x^\nu \geq 0.01$$

## Known Solution

$$\bar{x} = \bigl(0.29923815,\; \underbrace{0.06951128,\; \ldots,\; 0.06951128}_{9}\bigr)^T$$
