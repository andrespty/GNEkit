# Problem A7

A four-player GNEP with five variables per player and linear coupling constraints. The Jaco-Hessian is a **constant positive definite** $20 \times 20$ matrix.

**Players:** $N = 4$, each controlling $5$ variables. All variables bounded in $[1, 5]$.

## Objective Functions

$$\theta_\nu(\mathbf{x}) = \tfrac{1}{2}(x^\nu)^T A^\nu x^\nu + (x^\nu)^T B^\nu x^{-\nu}$$

The matrices $A^\nu$ and $B^\nu$ are recovered from the $20 \times 20$ Jaco-Hessian (block-diagonal entries give $A^\nu$; off-diagonal blocks give $B^\nu$). See the paper (Appendix A.7) for the full Jaco-Hessian.

## Constraints

**Bound constraints:** $1 \leq x^\nu_i \leq 5$ for all players and variables.

**Linear coupling constraints (one per player):**

$$x^1_1 + 2x^1_2 - x^1_3 + 3x^1_4 - 4x^1_5 \leq 2 - x^2_2 + 3x^2_3$$

$$-x^2_1 + 3x^2_2 - 2x^2_3 + x^2_4 + 3x^2_5 \leq 4 - x^3_1 + 3x^3_5 - 2x^4_3$$

$$-2x^3_1 + 3x^3_2 + x^3_3 - x^3_4 - 2x^3_5 \leq 4 - x^1_1 + 4x^4_5$$

$$4x^4_1 - 2x^4_2 - 3x^4_3 - 6x^4_4 + 5x^4_5 \leq 3 - x^1_1 - x^1_2 + x^2_1 + x^2_2$$

## Known Solution

All 20 components converge to approximately $1.0$ (near the lower bound), except:

$$x^2_5 \approx 1.843$$
