# Problem A14

The **internet switching model** introduced by Kesselman et al. (2005) and analyzed by Facchinei et al. (2009). All players are symmetric. The Jaco-Hessian is non-symmetric but positive definite at the solution.

**Players:** $N = 10$, each controlling a single variable $x^\nu \in \mathbb{R}$, with $B = 1$.

## Objective Functions

$$\theta_\nu(\mathbf{x}) := \frac{-x^\nu}{x^1 + \cdots + x^N}\left(1 - \frac{x^1 + \cdots + x^N}{B}\right), \quad \forall\, \nu = 1, \ldots, N$$

## Constraints

$$x^1 + \cdots + x^N \leq B, \quad x^\nu \geq 0.01 \quad \forall\, \nu$$

## Known Solution

$$\bar{x} = \left(\frac{9}{100}, \ldots, \frac{9}{100}\right)^T = (0.09,\; \ldots,\; 0.09)^T$$
