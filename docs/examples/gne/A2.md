# Problem A2

A variant of Problem A1 in which players 2–5 have a modified (squared) objective function and the feasible region is made very narrow by an additional lower bound on the total strategy.

**Players:** $N = 10$, each controlling a single variable $x^\nu \in \mathbb{R}$

## Objective Functions

Players 1, 6–10 share the objective from Problem A1:

$$\theta_\nu(\mathbf{x}) := \frac{-x^\nu}{x^1 + \cdots + x^N}\left(1 - \frac{x^1 + \cdots + x^N}{B}\right)$$

Players 2, 3, 4, 5 use a squared variant:

$$\theta_\nu(\mathbf{x}) := \frac{-x^\nu}{x^1 + \cdots + x^N}\left(1 - \frac{x^1 + \cdots + x^N}{B}\right)^2$$

with $B = 1$.

## Constraints

**Player 1:** $\quad 0.3 \leq x^1 \leq 0.5$

**Players 2–4, 7–10:** $\quad x^1 + \cdots + x^N \leq B,\quad x^\nu \geq 0.01$

**Players 5 and 6 (additional constraint):**

$$0.99 \leq x^1 + \cdots + x^N \leq B$$

**Player 9:** $\quad x^9 \leq 0.06$ $\quad$ **Player 10:** $\quad x^{10} \leq 0.05$

## Known Solution

Two solutions are found (starting points 0.01 and 0.1); the third starting point fails.

$$\bar{x}^{(1)} = (0.29963,\; 0.00998,\; 0.00998,\; 0.00998,\; 0.59852,\; 0.02187,\; 0.00999,\; 0.00999,\; 0.00999,\; 0.00999)^T$$
