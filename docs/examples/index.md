# Examples

This page collects example problems implemented in `GNEkit`. Each example
is taken directly from the GNEP literature and shows how to express the
problem using the library — from defining players and objectives to computing
equilibria.

The examples are organized by problem type and grow as the library develops.
If you have a problem from the literature you would like to see implemented,
feel free to open an issue or contribute.
